¿Qué es el Instrumento Virtual XYZ?

El Instrumento Virtual XYZ es una innovadora herramienta diseñada para medir y controlar de manera precisa los parámetros clave de un Mosfet, como tensión, corriente y temperatura. Equipado con controladores PID dedicados, este sistema asegura una regulación estable y precisa bajo cualquier condición de operación, proporcionando un control total sobre el dispositivo.

En su núcleo, el instrumento utiliza el microcontrolador Blue Pill (STM32F103C8T6) para la adquisición de datos, mientras que el procesamiento de la información y la interfaz gráfica se gestionan a través de software Qt, lo que permite una experiencia de usuario intuitiva y eficiente.

El Instrumento Virtual XYZ es compatible con PC, y puede conectarse fácilmente a Windows 10 o superior mediante un puerto USB y un conversor USB-UART. Se recomienda contar con al menos 8 GB de RAM.

Además, el sistema incluye una salida RS485 Half-Duplex, que utiliza el protocolo MODBUS RTU para facilitar la comunicación y el control. Las funciones principales incluyen Read Holding Registers para obtener señales analógicas y Write Holding Registers para controlar salidas analógicas.
